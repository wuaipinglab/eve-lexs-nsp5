# EVE (Evolutionary model of Variant Effects)

The `EVE/` directory under this folder is from the [OATML/EVE](https://github.com/OATML/EVE) repository.

## Required File Structure

To run the computational scripts of this project, at least the following files and directories are required:

- `EVE/compute_evol_indices.py`
- `EVE/train_VAE.py`
- `EVE/utils/` (contains `data_utils.py`, etc.)
- `EVE/EVE/` (contains `VAE_model.py`, `VAE_encoder.py`, `VAE_decoder.py`, `default_model_params.json`)

## How to Obtain

```bash
# Run in the src/ directory:
git clone https://github.com/OATML/EVE.git
# Or manually place the EVE directory here
```
